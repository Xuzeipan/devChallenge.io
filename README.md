# DevChallenges.io Projects

This repository contains a collection of programming challenges and projects from [DevChallenges.io](https://devchallenges.io/), a platform dedicated to helping developers improve their skills through real-world project challenges.

## About DevChallenges.io

DevChallenges.io is a learning platform that provides developers with practical, hands-on challenges to enhance their coding abilities. The challenges range from front-end design implementations to full-stack applications, helping developers build a strong portfolio of projects.

## Projects Included

This repository includes the following challenge projects:

### 1. Minimal Blog Card
A minimalist blog card component challenge that focuses on clean design and responsive layout. This project demonstrates HTML5, CSS3, and web design principles.

- **Location:** `Minimal-blog-card-starter-master/`
- **Challenge Resources:** Available in `design/` and `resources/` directories

### 2. Simple FAQ
A frequently asked questions component that implements interactive features and accordion functionality. This project covers HTML, CSS, and basic JavaScript interactivity.

- **Location:** `simple-faq-master/`
- **Challenge Resources:** Available in `design/` and `resources/` directories

## Getting Started

1. Navigate to the specific project folder
2. Open the `index.html` file in your browser to view the project
3. Review the `design/` folder for design specifications
4. Check the `resources/` folder for any additional assets

## Challenge Resources

All design files and resources for these challenges are sourced from [DevChallenges.io](https://devchallenges.io/). Each project folder contains:

- **Design files:** Visual mockups and specifications
- **Resources:** Images, icons, and other assets needed for the implementation
- **HTML/CSS:** Implementation files

## Learning Objectives

Through these challenges, the following skills are developed:

- HTML5 semantic markup
- CSS3 styling and responsive design
- Web component design principles
- User interface implementation
- Cross-browser compatibility
